# Chrome Web Store listing (draft)

**Title:** IOTA Names Resolver (.iota)  
**Short description:** Resolve *.iota names via IOTA JSON-RPC and optionally redirect to the configured website.  
**Detailed description:**
- Resolves *.iota names using iotax_iotaNamesLookup
- Shows records and metadata
- Optional auto-redirect when a website record is present
- Works around Browser “search vs URL” behavior via omnibox keyword `iota`
